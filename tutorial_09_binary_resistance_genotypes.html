Tutorial 09 — Binary Resistance Genotypes¶

Prerequisites: Tutorials 01–08

When using a phage cocktail of m species, the bacteria can resist each phage independently, producing 2^m distinct genotypes. Enumerating and parameterising all of these by hand is impractical. BinaryResistanceGenotypes (BRG) does this automatically.

What you will learn

  1. The BRG genotype ordering and labelling convention
  2. BacterialStrain and PhageStrain objects — the preferred constructor
  3. Automatic growth_rates, adsorption_rates, and mutation_matrix
  4. equilibrium_initial_condition — start at mutation-selection balance
  5. Phage–antibiotic synergy: evolutionary steering via differential Emax
  6. Inspecting what BRG computes before running simulations

2 Single-phage BRG (2 strains)¶

With 1 phage, BRG produces 2 genotypes:

Index Label Meaning
0 '0' WT — susceptible to phi0
1 '1' R — resistant to phi0

Fitness cost of resistance (fitness_cost): resistant strains typically grow slower because receptor loss comes with a metabolic penalty. Values of 2–10% per locus are commonly reported.

Mutation rate (mu): the per-replication rate at which a susceptible bacterium acquires resistance. Typical values: 10⁻⁸ – 10⁻⁶ h⁻¹ for spontaneous phage resistance mutations.

3 Two-phage cocktail BRG (4 strains)¶

With 2 phages, BRG enumerates 4 genotypes:

Index Label Meaning
0 '00' WT — susceptible to phi0 AND phi1
1 '01' R_phi1 — susceptible to phi0, resistant to phi1
2 '10' R_phi0 — resistant to phi0, susceptible to phi1
3 '11' R_both — resistant to both

Clinical rationale for a 2-phage cocktail: phage of different receptor specificities are less likely to be simultaneously evaded. However, using phages with independent resistance mechanisms prevents cross-resistance — a bacterium must acquire two independent mutations to escape the cocktail.

Cost of double resistance is multiplicative: a 6%-cost-per-locus bacterium with both resistances grows at μ × (1-0.06)² ≈ 0.88 × μ — a significant disadvantage that keeps the double-resistant minority rare in the absence of selective pressure.

4 equilibrium_initial_condition — starting at mutation–selection balance¶

Seeding only wild-type bacteria creates an artificial lag before resistant mutants appear. In reality, rare resistant variants pre-exist at the mutation–selection balance frequency:

freq_R ≈ μ / (μ + growth_cost)

equilibrium_initial_condition(total_bacteria) computes this distribution using replicator dynamics. Starting from balance produces more realistic early dynamics where resistant mutants are already present at low but non-zero frequencies.

5 Phage–antibiotic synergy: evolutionary steering¶

BRG antibiotic support: BinaryResistanceGenotypes supports antibiotic resistance loci via the Antibiotic object (per-strain Emax/EC50 for ABX-susceptible vs ABX-resistant genotypes). The optional phage_abx_ec50_factor[j, a] further adjusts the EC50 of antibiotic a in strains carrying phage-resistance at locus j.

Evolutionary steering requires completely non-overlapping kill profiles:

Strain Growth Phage adsorption ABX Emax Role
WT (0) 1.2 h⁻¹ 1×10⁻⁷ mL/h 0.5 (low) Phage-sensitive, ABX-insensitive
R (1) 1.0 h⁻¹ 1×10⁻¹¹ mL/h 4.0 (high) Phage-resistant, ABX-sensitive

With phage alone: phage kills WT → R expands unopposed. With ABX alone: R killed, but WT uninhibited (Emax too low). Combined: phage removes WT; antibiotic removes R → full clearance.

Because this requires per-strain Emax control (not just EC50 modulation), we use ModelBuilder directly rather than the BRG Antibiotic API here.

# BRG Antibiotic API — shown here for API reference only
from pbisim.strains.genotypes import Antibiotic
from pbisim import ModelBuilder

cipro_obj = Antibiotic(
    name="cipro",
    emax_s=5.0, emax_r=5.0,    # same Emax; ABX resistance raises EC50
    ec50_s=0.5, ec50_r=15.0,   # resistant: 30× higher EC50
    k_elim=0.5, Vc=15.0,
    fitness_cost=0.05,
    mu=1e-8,
)
brg_combo = BinaryResistanceGenotypes.from_strains(
    phages=[phi0], bacteria=bacteria, antibiotics=[cipro_obj],
    phage_abx_ec50_factor=np.array([[0.5]]),  # phi0-resistance lowers cipro EC50
)
print(f"BRG n_strains (2^(m=1+a=1) = 4): {brg_combo.n_strains}")
print("Strain labels:", brg_combo.strain_labels)
ec50_mat = brg_combo.antibiotic_ec50
print("Effective cipro EC50 per strain:")
for lbl, ec50_val in zip(brg_combo.strain_labels, ec50_mat[:, 0]):
    print(f"  {lbl}: {ec50_val:.3f} mg/L")

# Evolutionary steering demo — ModelBuilder with per-strain differential Emax
anti_cfg = (
    ModelBuilder(n_bacteria=2, n_phages=1, n_latent=5, n_depth=2)
    .with_growth_rates([1.2, 1.0], bacteria_to_resource_ratio=1e8)
    .with_mutations([[-1e-7, 0], [1e-7, 0]])        # WT → R at 1e-7 h⁻¹
    .with_phage_params(
        adsorption_rates         = [1e-7,  1e-11],  # (WT, R): phage kills WT only
        adsorption_rates_dormant = [1e-10, 1e-11],
        burst_sizes=50.0, latent_periods=0.5, phage_decay_rates=0.1,
    )
    .with_dormancy(dormancy_rate=1.0, resuscitation_rate=0.01)
    .with_antibiotic('cipro',
                     k_elim=0.6, Vc=1.0,            # Vc=1 → amount = concentration
                     emax=[0.5, 4.0],               # WT: Emax=0.5; R: Emax=4.0
                     ec50=[0.5, 0.5], hill=2.0)
    .with_nutrient(monod_constant=0.3, recycle_fraction=0.0)
)

# Fixed phage dose (1e4 PFU/mL at t=0); vary antibiotic dose q12h
fig, axes = plt.subplots(1, 3, figsize=(13, 4))
doses  = [0, 0.1, 0.5, 1.0]
colors = ["C0", "C1", "C2", "C3"]

for dose, color in zip(doses, colors):
    events = [DoseEvent(time=t, amount=dose, target="antibiotic", index=0)
              for t in [0, 12, 24, 36]]
    cfg = anti_cfg.with_dose_schedule(DoseSchedule(events)).build()
    res = solve_ode(
        PBIModel(cfg, initial_B=np.array([1e7, 1e3]),
                 initial_P=np.array([1e4]), initial_S=1.0),
        t_end=48.0, dt=0.25,
    )
    B_tot = res.sum_prefixes("B","D","I","H")
    P_tot = res.sum_prefixes("P")
    A_cc  = res.get("Ac0")           # Vc=1 → amount = Cc (mg/L)
    label = f"ABX {dose} mg/L q12h"
    axes[0].semilogy(res.time, np.maximum(B_tot, 1), color=color, label=label)
    axes[1].semilogy(res.time, np.maximum(P_tot, 1), color=color, label=label)
    axes[2].plot(res.time, np.maximum(A_cc,  0),     color=color, label=label)

axes[0].set(xlabel="Time (h)", ylabel="Cells/mL", title="Total bacteria")
axes[1].set(xlabel="Time (h)", ylabel="PFU/mL",   title="Free phage")
axes[2].set(xlabel="Time (h)", ylabel="mg/L",     title="Antibiotic Cc")
axes[2].axhline(0.5, ls=":", color="k", lw=0.8, label="EC50")
for ax in axes:
    ax.legend(fontsize=8)
plt.suptitle(
    "Evolutionary steering: phage clears WT (Emax_ABX=0.5); "
    "ABX clears resistant R (Emax_ABX=4.0)"
)
plt.tight_layout()

6 Summary¶

Property Value / Description
Genotype ordering cart_product([[0,1]]*m) — row = genotype, column = phage locus
Bit convention bit j = 1 means resistant to phage j
Label (phage only) '00', '01', '10', '11' for m=2
Label (with antibiotics) 'phi00_abx01' etc.
Fitness cost Multiplicative per locus: μ × Π(1 − cost[j])^G[j]
Mutation matrix Auto-generated irreversible DAG; mass-conserving
equilibrium_initial_condition Start at mutation–selection balance
BRG antibiotic loci antibiotics=[Antibiotic(...)] adds ABX-resistance loci; phage_abx_ec50_factor modulates EC50
Evolutionary steering Per-strain emax differences (via ModelBuilder) give orthogonal kill profiles
