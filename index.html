pbisim — Phage-Bacteria Interaction Simulation

Tutorial series for the pbisim Python package — PKPD modelling of phage therapy.

GitHub repository · Download notebooks (.ipynb)

Getting started

  1. Tutorial 01 — First Simulation — build, run and interpret a minimal phage-bacteria ODE model
  2. Tutorial 02 — Growth and Nutrients — Monod kinetics, nutrient depletion, carrying capacity
  3. Tutorial 03 — Dormancy — persistent bacteria, depth layers, nutrient-driven resuscitation
  4. Tutorial 04 — Immune Response — innate immune module, Hill killing, adaptive antibody response

Pharmacokinetics and pharmacodynamics

  1. Tutorial 05 — Phage Pharmacokinetics — IV dosing, effect-compartment vs mass-conserving mode, k_back spill-back
  2. Tutorial 06 — Antibiotic PK/PD — two-compartment PK, Hill-equation kill rates, combination therapy
  3. Tutorial 07 — Dosing Schedules — DoseEvent targets, bolus vs infusion, multi-dose regimens

Multi-strain dynamics and resistance

  1. Tutorial 08 — Multi-Strain Dynamics — StrainSet API, named clinical strains, per-strain antibiotic profiles
  2. Tutorial 09 — Binary Resistance Genotypes — BRG: automatic 2^m genotype enumeration, equilibrium ICs, steering
  3. Tutorial 10 — Cross-Resistance and Collateral Sensitivity — Type A (phenotypic) and Type B (genotypic) cross-resistance

Clinical trial simulation

  1. Tutorial 11 — IIV and Virtual Populations — LogNormal variability, VirtualPopulation, TrialRunner
  2. Tutorial 12 — Clinical Trials — ClinicalTrial matched-twins design, treatment arms, result.compare()

Advanced features

  1. Tutorial 13 — Advanced Features — superinfection, presets, show_equations, get_parameters, feature flags
