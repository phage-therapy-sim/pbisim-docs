2 Superinfection — phage adsorption to infected bacteria¶

By default, phage only adsorb to active uninfected bacteria (B) and dormant uninfected bacteria (D). Once a bacterium is infected (I or H), it is shielded from secondary phage infection — a simplification based on the concept of superinfection exclusion (where the first infecting phage blocks subsequent ones).

Setting allow_superinfection=True removes this shield: phage can also adsorb to I and H compartments. This is biologically relevant when:

  • Modelling phage cocktails where genetic recombination between co-infecting phages generates novel variants (directed evolution)
  • Studying lytic–lysogenic switching where double infection affects fate
  • Benchmarking against experimental data that shows phage loss beyond what B + D adsorption alone can explain

The secondary adsorption depletes the free-phage pool but does NOT create a new infected compartment (host fate is unchanged for simplicity).

In [2]:
# Two-strain model: strain 0 is phage-susceptible (adsorption=1e-8),
# strain 1 is phage-resistant (adsorption=0).  With superinfection enabled,
# phage also adsorb to already-infected I/H bacteria, depleting the phage
# pool faster — especially visible when many bacteria are in the infected state.
base_si = (
    ModelBuilder(n_bacteria=2, n_phages=1, n_latent=5)
    .with_growth_rates([1.2, 1.15], bacteria_to_resource_ratio=1e9)
    .with_phage_params(adsorption_rates=[1e-8, 0], burst_sizes=20.0,
                       latent_periods=2, phage_decay_rates=0.05)
    .with_nutrient(monod_constant=0.3, recycle_fraction=0.0)
    .with_dose_schedule(DoseSchedule([
        DoseEvent(time=0, amount=5e6, target="phage", index=0)
    ]))
)

fig, axes = plt.subplots(1, 2, figsize=(11, 4))
for (si, color, label) in [
    (False, "C0", "Superinfection excluded (default)"),
    (True,  "C2", "allow_superinfection=True"),
]:
    cfg = base_si.with_phage_params(allow_superinfection=si).build()
    res = solve_ode(
        PBIModel(cfg, initial_B=np.array([1e7, 1e3]),
                 initial_P=np.array([0.0]), initial_S=1.0),
        t_end=48.0, dt=0.25,
    )
    axes[0].semilogy(res.time,
                     np.maximum(res.sum_prefixes("B","D","I","H"), 1),
                     color=color, label=label)
    axes[1].semilogy(res.time, np.maximum(res.get("P0"), 1),
                     color=color, label=label)

axes[0].set(xlabel="Time (h)", ylabel="Cells/mL", title="Total bacteria")
axes[1].set(xlabel="Time (h)", ylabel="PFU/mL",
            title="Free phage — superinfection depletes phage faster")
for ax in axes: ax.legend(fontsize=9)

3 Dynamic transition function — phage-pressure-induced resistance switching¶

with_transition_function(fn) adds a state-dependent phenotypic switch on top of the static transition_rates matrix. The function is evaluated at every ODE time step and its output is ADDED to the static matrix.

Biological application: phage stress triggers upregulation of resistance mechanisms in some bacteria (e.g., CRISPR adaptation, OM protein downregulation). When phage density is high, susceptible bacteria switch faster to a resistant phenotype.

In [3]:
def phage_driven_switch(S, B, P, config):
    # Phage-pressure-dependent rate of switching susceptible -> resistant
    n = config.n_bacteria
    m = config.n_phages
    phage_pressure = float(np.sum(P)) / (1e6 + float(np.sum(P)))  # saturating signal
    k_switch = 0.05 * phage_pressure   # max switching rate 0.05 h^-1
    T = np.zeros((n, n))
    if n >= 2:
        T[1, 0] =  k_switch   # S -> R rate increases with phage
        T[0, 0] = -k_switch   # S loses bacteria at same rate
    return T

cfg_switch = (
    ModelBuilder(n_bacteria=2, n_phages=1, n_latent=5)
    .with_growth_rates([1.0, 0.92])
    .with_phage_params(
        adsorption_rates=np.array([[2e-9], [2e-11]]),
        adsorption_rates_dormant=np.array([[2e-11], [2e-13]]),
        burst_sizes=50.0, latent_periods=0.5, phage_decay_rates=0.1,
    )
    .with_transition_function(phage_driven_switch)  # dynamic switch
    .with_nutrient(monod_constant=0.3, recycle_fraction=0.0)
    .with_dose_schedule(DoseSchedule([
        DoseEvent(time=0, amount=5e8, target="phage", index=0)
    ]))
    .build()
)

res_switch = solve_ode(
    PBIModel(cfg_switch, initial_B=np.array([5e7, 1e3]),
             initial_P=np.array([0.0]), initial_S=1.0),
    t_end=72.0, dt=0.25,
)

fig, ax = plt.subplots(figsize=(8, 4))
for i, (label, color) in enumerate([("Susceptible (S)", "C0"),
                                     ("Resistant (R)",   "C2")]):
    ax.semilogy(res_switch.time, np.maximum(res_switch.get(f"B{i}"), 1),
                color=color, label=label)
ax.set(xlabel="Time (h)", ylabel="Cells/mL",
       title="Phage-pressure-driven phenotypic switching S→R")
ax.legend()
plt.tight_layout()

4 Growth/dormancy presets¶

Presets bundle a consistent set of growth, dormancy, and resuscitation functions so you do not need to manually verify their compatibility.

Preset Growth Dormancy trigger track_nutrients
CONSTANT Constant (no limit) Constant rate False
LOGISTIC Logistic (density) Density-dependent False
MONOD Monod (nutrient) Nutrient-dependent True
MONOD_LOGISTIC Monod × logistic Nutrient + density True

apply_preset(cfg, preset) returns a new ModelConfig with the preset's functions applied. It also sets track_nutrients correctly.

In [4]:
presets = [CONSTANT, LOGISTIC, MONOD, MONOD_LOGISTIC]
labels  = ["CONSTANT", "LOGISTIC", "MONOD", "MONOD_LOGISTIC"]

fig, ax = plt.subplots(figsize=(9, 4))

for preset, label, color in zip(presets, labels, ["C0","C1","C2","C3"]):
    # Start from a minimal config
    kw = {"carrying_capacity": 1e9} if label in ("LOGISTIC","MONOD_LOGISTIC") else {}
    cfg_p = (
        ModelBuilder(n_bacteria=1, n_phages=0)
        .with_growth_rates(1.0, bacteria_to_resource_ratio=1e9)
        .with_dormancy(dormancy_rate=0.1, resuscitation_rate=0.05)
        .with_nutrient(monod_constant=0.3, **kw,
                       track_nutrients=preset in (MONOD, MONOD_LOGISTIC))
        .build()
    )
    cfg_p = apply_preset(cfg_p, preset)
    res_p = solve_ode(
        PBIModel(cfg_p, initial_B=np.array([1e4]),
                 initial_P=np.zeros(0), initial_S=1.0),
        t_end=24.0, dt=0.1,
    )
    ax.semilogy(res_p.time, np.maximum(res_p.get("B0"), 1),
                color=color, label=label)

ax.set(xlabel="Time (h)", ylabel="Cells/mL",
       title="Preset growth functions — all four built-in presets")
ax.legend()
plt.tight_layout()
plt.show()

5 show_equations and get_parameters — model introspection¶

Two tools help you verify that the model you built is what you intended:

  • model.show_equations(): renders the full ODE system as LaTeX in Jupyter, or prints raw LaTeX in a terminal. Flags which features are active (phage PK, dormancy, antibiotic PD, etc.).

  • model.get_parameters(): returns all current parameter values as a flat dictionary — useful for sanity-checking that broadcasting and defaults worked correctly before a long simulation run.

In [5]:
# Build a model with several features active
cfg_inspect = (
    ModelBuilder(n_bacteria=2, n_phages=1, n_latent=3, n_depth=2)
    .with_growth_rates([1.0, 0.93], bacteria_to_resource_ratio=1e9)
    .with_phage_params(adsorption_rates=np.array([[2e-9],[2e-11]]),
                       burst_sizes=50.0, latent_periods=0.5,
                       phage_decay_rates=0.1)
    .with_dormancy(dormancy_rate=0.1, resuscitation_rate=0.05,
                   dormancy_signal="nutrient", resuscitation_signal="nutrient")
    .with_immunity(immune_module="innate", imm_stim_rate=1.0,
                   imm_kill_rate=3e6, imm_decay_rate=0.15,
                   imm_stim50=5e6, imm_kill50=5e6)
    .with_mutations(phage_resistance_rates=[5e-8])
    .with_nutrient(monod_constant=0.3, recycle_fraction=0.3)
    .build()
)
model_inspect = PBIModel(cfg_inspect,

ODE System (LaTeX source):

\begin{aligned}
  \dfrac{dB_i}{dt} &= \mu_i\,f(S)\,B_i && \text{\small growth} \\
    &\quad +\; \textstyle\sum_k M_{ik}\,B_k && \text{\small mutations from/to other strains} \\
    &\quad -\; \varphi_i\,B_i && \text{\small entry into dormancy} \\
    &\quad +\; \rho_i\,D_{0,i} && \text{\small resuscitation} \\
    &\quad -\; \textstyle\sum_j \beta_{ij}\,P_j\,B_i && \text{\small phage adsorption} \\
    &\quad -\; \frac{\kappa_i\,\mathrm{Imm}\,B_i}{K_\mathrm{kill}+B_\mathrm{tot}} && \text{\small immune killing} \\\\
  \dfrac{dP_j}{dt} &= \textstyle\sum_i N_{ij}\,\gamma_{ij}\,I_{3,i,j}\,\phi && \text{\small burst lysis from active infected (last latent stage)} \\
    &\quad +\; \textstyle\sum_{d,i} N_{ij}\,\gamma_{ij}^D\,H_{d,3,i,j}\,\phi && \text{\small burst lysis from hibernating bacteria} \\
    &\quad -\; \textstyle\sum_i \beta_{ij}\,B_i\,P_j && \text{\small adsorption to active bacteria} \\
    &\quad -\; \textstyle\sum_{d,i}\beta_{ij}^D e^{-\alpha_{ij}d}\,D_{d,i}\,P_j && \text{\small adsorption to dormant bacteria} \\
    &\quad -\; \delta_j\,P_j && \text{\small phage decay} \\\\
  \dfrac{dI_{l,i,j}}{dt} &= \beta_{ij}\,B_i\,P_j\;\mathbf{1}_{[l=1]} && \text{\small new infection (first stage only)} \\
    &\quad -\; \gamma_{ij}\,I_{l,i,j}\,\phi && \text{\small progression / lysis at last stage} \\
    &\quad +\; \gamma_{ij}\,I_{l-1,i,j}\,\phi\;\mathbf{1}_{[l>1]} && \text{\small arrival from previous stage} \\
    &\quad -\; \varphi_i\,I_{l,i,j} && \text{\small dormancy entry} \\
    &\quad +\; \rho_i\,H_{0,l,i,j} && \text{\small resuscitation from hibernation} \\
    &\quad -\; \frac{\kappa_i\,\mathrm{Imm}\,I_{l,i,j}}{K_\mathrm{kill}+B_\mathrm{tot}} && \text{\small immune killing} \\\\
  \dfrac{dD_{d,i}}{dt} &= \varphi_i\,B_i\;\mathbf{1}_{[d=1]} && \text{\small dormancy entry (surface layer)} \\
    &\quad -\; \rho_i\,D_{0,i}\;\mathbf{1}_{[d=1]} && \text{\small resuscitation (surface layer only)} \\
    &\quad -\; \xi_i\,(1-f_S)\,D_{d,i} && \text{\small diffusion to deeper layer} \\
    &\quad +\; \xi_i\,f_S\,D_{d+1,i}\;\mathbf{1}_{[d<Q]} && \text{\small diffusion toward surface} \\
    &\quad -\; \textstyle\sum_j\beta_{ij}^D e^{-\alpha_{ij}d}\,D_{d,i}\,P_j && \text{\small phage infection of dormant bacteria} \\\\
  \dfrac{d\mathrm{Imm}}{dt} &= \frac{\textstyle\sum_i \sigma_i\bigl(B_i + \textstyle\sum_{l,j}I_{l,i,j}\bigr)}{K_\mathrm{stim}+B_\mathrm{tot}} && \text{\small immune stimulation} \\
    &\quad -\; \delta_\mathrm{Imm}\,\mathrm{Imm} && \text{\small immune decay} \\\\
  \dfrac{dS}{dt} &= \textstyle\sum_i\frac{\mu_i f(S)\,B_i}{Y_i} && \text{\small nutrient consumption by growing bacteria} \\
    &\quad +\; \eta\,\textstyle\sum_i\frac{\text{deaths}_i}{Y_i} && \text{\small biomass recycling from all death events}
\end{aligned}

State variables:
  B_i                             active bacteria, strain i = 1 … 2
  P_j                             free phage, species j = 1 … 1
  I_{l,i,j}                       active infected, latent stage l = 1 … 3
  D_{d,i}                         dormant bacteria, depth d = 1 … 2
  H_{d,l,i,j}                     dormant + infected (hibernating)
  \mathrm{Imm}                    immune effector concentration
  S                               nutrient substrate

Parameters:
  \mu_i                           maximum growth rate (h⁻¹)
  f(S)                            growth signal ∈ [0, 1]
  Y_i                             bacteria-to-resource ratio
  K_s                             Monod half-sat. = 0.3
  M_{ik}                          mutation rate from strain k → i per replication
  \beta_{ij}                      adsorption rate of phage j to strain i (mL h⁻¹)
  N_{ij}                          burst size — phage released per lysis event
  \gamma_{ij}                     latent-stage progression rate = (L+1) / τ_{ij}
  \tau_{ij}                       mean latent period of active infected (h)
  \delta_j                        phage decay / clearance rate (h⁻¹)
  \phi                            lysis progression signal (= f(S) by default)
  \varphi_i                       dormancy entry rate (h⁻¹)
  \rho_i                          resuscitation rate (h⁻¹)
  \xi_i                           dormancy-depth diffusion rate constant (h⁻¹)
  f_S                             nutrient signal for depth diffusion = S / (K_s + S)
  \beta_{ij}^D                    adsorption rate to dormant bacteria
  \alpha_{ij}                     depth attenuation of phage penetration
  \sigma_i                        immune stimulation rate per bacterium
  \kappa_i                        immune kill rate coefficient
  K_\mathrm{stim}                 immune stimulation half-sat. = 5e+06
  K_\mathrm{kill}                 immune killing half-sat. = 5e+06
  \delta_\mathrm{{Imm}}           immune effector decay rate = 0.15
  \eta                            biomass recycle fraction = 0.3

6 Quick-reference: when to use each builder¶

Modelling goal                          → Recommended builder
──────────────────────────────────────────────────────────────
Single strain, prototype / tutorial     → ModelBuilder
Named clinical strains, per-strain abx  → StrainSet + StrainDefinition
All 2^m phage-resistance genotypes      → BinaryResistanceGenotypes.from_strains()
Quick 2-strain binary-resistance        → ModelBuilder + phage_resistance_rates=[μ]
