Tutorial 07 — Dosing Schedules¶

Prerequisites: Tutorials 01–06

Tutorials 05 and 06 introduced DoseEvent within specific PK contexts. This tutorial covers the unified dose schedule comprehensively: all four targets, bolus vs infusion, clinical multi-dose regimens, and combination phage–antibiotic dosing for synergistic therapy.

What you will learn

  2. Bolus vs continuous infusion
  3. Multiple-dose antibiotic regimens (q12h, q8h, loading dose)

1 Imports and shared model¶

import numpy as np
import matplotlib.pyplot as plt

from pbisim import ModelBuilder, PBIModel, solve_ode, DoseEvent, DoseSchedule
from pbisim.analysis.single import plot_simulation

# Single-strain model with both phage (no PK) and antibiotic
def _base_cfg(phage_decay=0.1, abx_emax=4.0, abx_ec50=0.5):
    return (
        ModelBuilder(n_bacteria=1, n_phages=1, n_latent=5, n_depth=2)
        .with_growth_rates(1.0, bacteria_to_resource_ratio=1e9)
        .with_phage_params(adsorption_rates=2e-9, burst_sizes=50.0,
                           latent_periods=0.5, phage_decay_rates=phage_decay,
                           adsorption_rates_dormant=2e-11)
        .with_dormancy(dormancy_rate=0.15, resuscitation_rate=0.08,
                       dormancy_signal="nutrient", resuscitation_signal="nutrient")
        .with_antibiotic("cipro", k_elim=0.5, Vc=15.0, k12=0.4, k21=0.1,
                         emax=abx_emax, ec50=abx_ec50, hill=1.0,
                         emax_dormant=0.8, ec50_dormant=2.0)
        .with_nutrient(monod_constant=0.3, recycle_fraction=0.0)
    )

B0 = np.array([5e7])   # moderate inoculum
T_END = 72.0           # h

2 DoseEvent targets¶

The four dose targets and what they add to:

target State variable modified Typical use
"phage" P (no PK) or Pc (with PK) Phage bolus or infusion
"antibiotic" Ac[index] (amount in central) IV or oral antibiotic
"nutrient" S Chemostat nutrient addition
"bacteria" B[index] Inoculation, challenge models

amount is the quantity added instantaneously (bolus). For infusions, use route="infusion" with a duration (hours): DoseEvent(time=t_start, amount=total_amount, route="infusion", duration=hours, ...).

The four targets operate on different timescales, so they are best demonstrated in two focused simulations below.

2a "phage" and "antibiotic" targets¶

sched_phage_abx = DoseSchedule([

2b "nutrient" and "bacteria" targets¶

The "nutrient" target adds directly to S. Its effect is most visible when bacteria have consumed most of the available nutrient and entered dormancy — a nutrient spike then resuscitates dormant bacteria, creating a second growth wave.

The "bacteria" target adds directly to B[index], modelling a secondary inoculation or challenge.

Here we run a bacteria-only simulation (no phage or antibiotic) starting with low initial nutrient (S₀ = 0.3) to demonstrate both targets:

3 Bolus vs continuous infusion¶

Bolus (single amount): maximum peak concentration, short exposure. Favoured for concentration-dependent antibiotics (fluoroquinolones, aminoglycosides) where high peak/MIC ratio predicts outcome.

Infusion (continuous rate over an interval): sustained plateau concentration. Favoured for time-dependent antibiotics (beta-lactams) where time above MIC drives efficacy.

For phage, infusion models a continuous vascular drip — clinically relevant for severe bacteraemia where phage elimination is rapid.

In [4]:
# Compare: 400mg bolus vs 400mg infused over 4h (same total dose)
bolus_sched  = DoseSchedule([DoseEvent(time=0, amount=400.0, target="antibiotic", index=0)])
# 400 mg total infused over 4 h  (rate = 400/4 = 100 mg/h internally)
infuse_sched = DoseSchedule([DoseEvent(time=0, amount=400.0,
                                        route="infusion", duration=4.0,
                                        target="antibiotic", index=0)])

results_route = {}
for name, sched in [("Bolus 400 mg", bolus_sched),
                    ("Infusion 100 mg/h × 4 h", infuse_sched)]:
    cfg = _base_cfg().with_dose_schedule(sched).build()
    res = solve_ode(
        PBIModel(cfg, initial_B=B0, initial_P=np.array([0.0]), initial_S=1.0),
        t_end=T_END, dt=0.25,
    )
    results_route[name] = res

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
colors = ["C0", "C2"]
for (name, res), color in zip(results_route.items(), colors):
    axes[0].semilogy(res.time, np.maximum(res.get("B0"), 1),  color=color, label=name)
    Cc = res.get("Ac0") / 15.0
    axes[1].plot(res.time, np.maximum(Cc, 0), color=color, label=name)
axes[1].axhline(0.5, ls=":", color="k", lw=0.8, label="EC50")
axes[0].set(xlabel="Time (h)", ylabel="Cells/mL", title="Bacteria")
axes[1].set(xlabel="Time (h)", ylabel="mg/L", title="Central concentration Cc")
for ax in axes: ax.legend(fontsize=9)
plt.suptitle("Bolus vs infusion — same total dose (400 mg)")
plt.tight_layout()
plt.show()

4 Clinical antibiotic regimens¶

Real clinical regimens use multiple doses separated by fixed intervals. A loading dose is often given first to rapidly achieve therapeutic concentrations, followed by smaller maintenance doses.

The dose interval should be chosen so that concentrations stay above the MIC for a sufficient fraction of the dosing interval (time-dependent drugs) or peak concentrations remain high (concentration-dependent drugs).

In [5]:
def _abx_only(doses_mg_times):
    sched = DoseSchedule([
        DoseEvent(time=t, amount=mg, target="antibiotic", index=0)
        for t, mg in doses_mg_times
    ])
    cfg = _base_cfg().with_dose_schedule(sched).build()
    return solve_ode(
        PBIModel(cfg, initial_B=B0, initial_P=np.array([0.0]), initial_S=1.0),
        t_end=T_END, dt=0.25,
    )

regimens = {

    "q8h  200mg (frequent)":    [(t,200) for t in range(0, 72, 8)],
    "Loading 600mg + q12h 300mg": [(0,600)]+[(t,300) for t in range(12,72,12)],
}

fig, axes = plt.subplots(1, 2, figsize=(12, 4))

5 Phage–antibiotic combination timing¶

Phage-antibiotic synergy (PAS) is well documented: phage kill replicating bacteria, lysing them and releasing sub-inhibitory antibiotic concentrations into the local environment; antibiotics then mop up dormant bacteria and resistant minority populations.

Key timing questions:

  • Simultaneous vs phage-first vs antibiotic-first
  • Should phage be re-dosed?

Expected biology:

  • Phage first → bacteria suppressed before antibiotic → lower mutant selection
  • Antibiotic first → partially inhibits bacteria (reduced burst?) → fewer phage targets
  • Simultaneous → synergistic killing if non-antagonistic
  • Phage re-dose → sustains phage pressure after blood clearance

def _combo_run(phage_times, abx_times_mg):
    events = []
    for t in phage_times:
        events.append(DoseEvent(time=t, amount=1e8, target="phage", index=0))
    for t, mg in abx_times_mg:
        events.append(DoseEvent(time=t, amount=mg, target="antibiotic", index=0))
    sched = DoseSchedule(events)
    cfg = _base_cfg().with_dose_schedule(sched).build()
    return solve_ode(
        PBIModel(cfg, initial_B=B0, initial_P=np.array([0.0]), initial_S=1.0),
        t_end=T_END, dt=0.25,
    )

6 Summary¶

Task Code pattern
Bolus at t DoseEvent(time=t, amount=X, target=..., index=i)
Infusion t→t2 DoseEvent(time=t, end_time=t2, rate=R, target=..., index=i)
Multi-dose schedule DoseSchedule([ev1, ev2, ...]) with any mix of targets
Attach to model .with_dose_schedule(schedule) on ModelBuilder

Practical tips:

  • For antibiotics given as amount, the initial central concentration is amount / Vc (same units as ec50).
  • For phage given as amount without PhagePKConfig, amount is in PFU/mL (concentration at the site). With PhagePKConfig, it is phage count added to blood (Pc).
  • Re-dosing phage at 24–48 h replenishes the blood reservoir after initial clearance and is recommended when bacterial density remains high.
