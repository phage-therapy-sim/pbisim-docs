Tutorial 10 — Cross-Resistance and Collateral Sensitivity¶

Prerequisites: Tutorials 01–09

When two phages share structural similarities (e.g., the same host receptor), resistance to one may automatically confer resistance to the other — cross-resistance. The opposite is also possible: bacteria that evolve resistance to one phage become more susceptible to a second — collateral sensitivity. Both phenomena have direct clinical implications for phage cocktail design.

What you will learn

  1. Type A cross-resistance: phenotypic, scaled by cross_resistance (adsorption rates)
  2. Type A collateral sensitivity: cross_resistance[i,j] > 1.0
  3. Type B cross-resistance: genotypic, scaled by cross_resistance_prob (mutation routing)
  4. Combining both mechanisms

T_END = 24.0

bacteria = BacterialStrain(base_growth_rate=1.0, bacteria_to_resource_ratio=1e9)

def _phi(name, adsorption_s=2e-9, mu=5e-8, fitness_cost=0.06, decay_rate=0.1):
    return PhageStrain(
        name=name,
        adsorption_s=adsorption_s, adsorption_r=0.0,
        adsorption_dormant_s=adsorption_s*0.01,
        adsorption_dormant_r=0.0,
        burst_size_s=50.0, burst_size_r=40.0,
        latent_period_s=0.5, latent_period_r=0.5,
        latent_period_dormant_s=1.0, latent_period_dormant_r=1.0,
        fitness_cost=fitness_cost, mu=mu, decay_rate=decay_rate,
    )

# Higher mutation rates and distinct adsorption rates to make resistance
# dynamics clearly visible over a short (24 h) observation window.
phi0 = _phi("phi0", adsorption_s=2e-8, mu=5e-4)
phi1 = _phi("phi1", adsorption_s=1e-9,  mu=3e-4)

def _brg_to_sim(brg, phage_doses, B0=None, t_end=T_END):
    cfg = brg.to_config(n_latent=5, n_depth=1,
                        monod_constant=0.3, recycle_fraction=0.0)
    events = [DoseEvent(time=0, amount=d, target="phage", index=j)
              for j, d in enumerate(phage_doses)]
    cfg_d = dataclasses.replace(cfg, dose_schedule=DoseSchedule(events))
    if B0 is None:
        B0 = brg.equilibrium_initial_condition(5e7)
    return solve_ode(
        PBIModel(cfg_d, initial_B=B0, initial_P=np.zeros(len(phage_doses)),
                 initial_S=1.0),
        t_end=t_end, dt=0.25,
    )

2 Type A cross-resistance — shared receptor¶

cross_resistance[i, j] scales the adsorption rate of phage j in strains carrying resistance to phage i.

  • = 1.0 (default): independent receptors, no cross-resistance
  • < 1.0: partial cross-protection (e.g., common receptor for phi0 and phi1)
  • > 1.0: collateral sensitivity — resisting phi0 exposes an alternative receptor that phi1 exploits more effectively

Biological example: resistance to phage phi0 via receptor mutation also disrupts a surface protein that phi1 normally only weakly binds, but after the structural change phi1 binds 3× more efficiently.

3 Type B cross-resistance — genotypic linkage¶

cross_resistance_prob[i, j] is the probability that acquiring resistance to phage i simultaneously confers resistance to phage j in one mutational event.

Biological example: the phi0-receptor gene and the phi1-receptor gene are co-located in an operon. Mutations that truncate phi0-receptor binding frequently co-truncate phi1-receptor binding — linked acquisition.

This is biologically different from Type A:

  • Type A: existing resistant strains have altered susceptibility to another phage
  • Type B: the act of acquiring resistance to one phage probabilistically provides simultaneous resistance to another

# Symmetric CRP: acquiring resistance to either phage can co-confer the other.
# The two off-diagonal probabilities need NOT be equal — setting
# CRP[0,1] != CRP[1,0] models path-dependent cross-resistance where
# the mutation route phi0→phi1 has different linkage than phi1→phi0.

    (0.7, "C2", "Strong linkage (p=0.7, symmetric)"),

# Q: Does the API support Type B *collateral sensitivity*?
# A: No — cross_resistance_prob only routes mutations toward MORE resistant
#    states (0→1 bit flips in the genotype).  There is no mechanism in BRG
#    to make a bacterium MORE susceptible via mutation routing.
#    The phenotypic outcome of genotypic collateral sensitivity is captured
#    by Type A (cross_resistance[i,j] > 1.0), which raises the adsorption
#    rate of resistant strains — functionally identical in the ODE regardless
#    of whether the underlying mechanism is Type A or B.
print("Type B collateral sensitivity: not supported as mutation routing.")
print("Use cross_resistance[i,j] > 1.0 (Type A) to model the phenotypic outcome.")

4 Combining Type A collateral sensitivity + Type B cross-resistance¶

The two mechanisms are independent and can co-exist. Here we model a 2-phage cocktail where:

  • Type A (collateral sensitivity): R_phi0 and R_phi1 strains each become more susceptible to the other phage (favourable for the cocktail)
  • Type B (cross-resistance): acquiring R_phi0 has a 30% chance of simultaneously conferring R_phi1 (unfavourable — speeds double resistance)

The net outcome depends on which effect dominates.

5 Summary¶

Mechanism API parameter Effect
Type A cross-resistance cross_resistance[i,j] < 1.0 Carrying R_phi_i reduces phi_j susceptibility
Type A collateral sensitivity cross_resistance[i,j] > 1.0 Carrying R_phi_i increases phi_j susceptibility
Type B cross-resistance cross_resistance_prob[i,j] > 0 Acquiring R_phi_i also confers R_phi_j with probability p

Clinical insight: A phage cocktail designed with collateral sensitivity (Type A) between its members ensures that any single-resistance escape mutant is MORE susceptible to the second phage — eliminating the escape route. When combined with Type B cross-resistance (section 4), the net outcome depends on which mechanism dominates: strong collateral sensitivity can outweigh the acceleration of double-resistance acquisition.
